Task_3 三栏式布局五种解决办法

  • 浮动布局
    1. 优点：兼容性极好
    2. 缺点：必须要左右中的顺序组织HTML代码，且浮动元素脱离文档流，需要做清除浮动，防止高度塌陷等问题
    3. 兼容性：IE6+, Firefox 2+, Chrome 1+
  • 绝对定位布局
    1. 优点：兼容性极好，设置方便，且不容易出现问题
    2. 缺点：绝对定位元素脱离文档流，下面的所有子元素也会脱离文档流，故可用性和有效性较差
    3. 兼容性：IE6+, Firefox 2+, Chrome 1+
  • flexbox布局
    1. 优点：实现效果完美，适合移动端
    2. 缺点：PC端兼容性较差，不兼容IE10以下浏览器
    3. 兼容性：IE 10+, Safari3.1+, Firefox 2+, Chrome 4+, Opera 12.1+
  • 表格布局
    1. 优点：可以做三栏由中间高度撑起来的效果，实现简单
    2. 缺点：不适用左右固定高度的布局，PC端不兼容IE6/7
    3. 兼容性：IE 8+, Safari3.1+, Firefox 3+, Chrome 4+, Opera 10.1+
  • 网格布局
    1. 优点：实现极为简单
    2. 缺点：作为最新的解决方案，兼容性极差，只有各浏览器较新的版本才支持
    3. 兼容性：IE 10+, Safari10.1+, Firefox 52+, Chrome 57+, Opera 44+
